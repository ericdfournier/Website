  • Retina Ready

    Ultra Sharp Graphics

    We included java file (retina.js) in this template to support devices with retina display. When your users load a page, retina.js checks each image on the page to see if there is a high-resolution version of that image on your server. If a high-resolution variant exists, the script will swap in that image in-place. For example, if you have an image on your page that looks like this: /img/my_image.png"
    The script will check your server to see if an alternative image exists at this path: "/img/my_image@2x.png"
    Simplified, all you have to do is to include same image in double size with @2x added in file name, and in same folder as original image on your server. If you don't include double sized image then script will leave original one. I.E. In preview, there is no double sized images for section header backgrounds.

Parallax Effect

Works layout 1 + carousel with vertical slide effect

Parallax scrolling is an interesting technique, where, as you scroll, the background images translate slower/faster than the content in the foreground, creating the illusion of 3D depth. On mobile and tablet devices parallax is turned off for better perfomance. Control parallax background in header sections where you see /data-stellar-background-ratio="0.6"/. The ratio is relative to the natural scroll speed, so a ratio of 0.5 would cause the element to scroll at half-speed, a ratio of 1 would have no effect, and a ratio of 2 would cause the element to scroll at twice the speed. Visit script site for more settings.
